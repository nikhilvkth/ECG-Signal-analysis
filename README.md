# ECG Exploration Project

## Overview
This project explores ECG (Electrocardiogram) signals using Python and the MIT-BIH Arrhythmia Dataset from PhysioNet.

The purpose of this project is to:
- Load real biomedical ECG data
- Visualize ECG waveforms
- Understand heartbeat annotations
- Build foundational skills for cardiac AI research

---

## Dataset
Dataset used:
- MIT-BIH Arrhythmia Database
- Source: PhysioNet

---
